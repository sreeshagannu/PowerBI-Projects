# Basic Visualization

## Overview
This folder contains beginner-level Power BI reports focused on creating basic visualizations and understanding data representation.

## Concepts Covered
- Creating basic charts (bar, line, pie)
- Using maps for geographical data
- Category and subcategory analysis
- Building simple dashboards

## Files Included
- MyFirstVisualization.pbix – Introduction to basic visuals
- Charts and Maps.pbix – Combination of charts and map visuals
- Maps.pbix – Focus on geographical data visualization
- CategoryAndSubcategory.pbix – Analysis using categories and subcategories

## Tools Used
- Power BI

## Purpose
To build a strong foundation in data visualization and understand how to present data effectively using Power BI.
