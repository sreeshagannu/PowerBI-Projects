# DAX & Advanced Concepts

## Overview
This folder contains Power BI reports demonstrating advanced features and DAX concepts.

## Concepts Covered
- Quick measures
- Matrix visuals
- Drill-through functionality
- Date and time calculations

## Tools Used
- Power BI (DAX)

## Purpose
To understand advanced analytics and calculations in Power BI.
