# IT Ticket Analysis Dashboard

## Overview
This project analyzes IT support tickets to identify trends and performance.

## Key Insights
- Ticket volume by category and priority
- SLA compliance
- Resolution time trends

## Features
- Interactive filters
- KPI indicators
- Drill-through analysis

## Tools Used
- Power BI
